# Customer Support Ticket Triage System

## Project Description
This project is part of Week-1 Internship.
It helps companies organize customer support messages by identifying
the issue type, assigning priority, and calculating response time.

## Tools Used
- Python
- Google Colab
- GitHub

## Features
- Cleans customer messages
- Classifies issues (Payment, Login, Delivery, etc.)
- Assigns priority levels (P0 to P3)
- Calculates SLA response time
- Generates final report
